📜 Overview

Al-Maqlub (المقلوب, "The Reversed") is a single-file browser application dedicated to the ancient Arabic art of palindrome poetry. It brings together four experiences: a curated museum of classical palindromes, a real-time composition studio, an interactive puzzle game, and a dual-direction reciter.


Arabic palindrome poetry is one of the most technically demanding forms in the Arabic literary tradition. Known as المقلوب الحرفي (letter reversal), السرطاني (the crab poem), and other names, these compositions read identically — or produce new meaning — when reversed.


Al-Maqlub is part of the كلامي (Kalami) Arabic tools family, sharing the same design language: dark void background, gold accents, Amiri font, Islamic geometric patterns, and Bismillah header.

🗺️ How To Use

GETTING STARTED

1

Download

Get almaqlub.html (or the full zip). That's the only file you need.

2

Open in Browser

Double-click the file to open it in Chrome or Edge. No server needed, no installation, no build step.

3

Navigate

Click any of the four tabs — المكتبة (Library), المحرر (Creator), اللعبة (Game), المنشد (Reciter) — to switch sections.

USING THE LIBRARY

1

Browse

Scroll through the decorated poem cards. Each shows the Arabic text, type badge, translations, and source.

2

Analyze

Click تحليل to trigger the symmetry animation — matching mirror pairs light up gold from center outward, mismatches glow crimson.

3

Reverse

Click عكس to flip the text backwards and see the palindrome property in action.

4

Listen

Click استمع to hear the poem read aloud in Arabic.

COMPOSING PALINDROMES

1

Type or Dictate

Use your system keyboard, the on-screen Arabic keyboard, or click تسجيل to dictate via microphone.

2

Watch the Analysis

The panel updates in real time: letter/word palindrome checks, symmetry score arc, and the color-coded letter map.

3

Fix Mismatches

Crimson letters in the map show mismatched mirror positions. The suggestions panel tells you exactly which positions need correction.

4

Save or Export

Click حفظ to save to your session collection, or تصدير SVG to download a decorative card image.

PLAYING THE GAME

1

Choose Difficulty

حروف (fill 1–3 letters), كلمات (fill missing words), or بيت (reconstruct the mirror half).

2

Solve the Puzzle

Read the poem with blanks, type your answer, and click تحقق to check. Gold glow = correct, crimson shake = wrong.

3

Build Your Streak

Keep answering correctly to increase your streak. Enable the ⏱ timer for extra challenge.

❓ Frequently Asked Questions

GENERAL

Q: Do I need an internet connection?▶
Only for loading Google Fonts (Amiri, Cinzel, DM Mono) and for text-to-speech. The core palindrome logic, keyboard, game, and analysis all work offline. If fonts fail to load, the browser uses fallback serif/monospace fonts.
Q: Which browser should I use?▶
Chrome or Edge for the full experience (speech recognition + synthesis). Firefox supports TTS but not mic input. Safari has partial support.
Q: Can I use this on mobile?▶
Yes. The layout is fully responsive. The on-screen Arabic keyboard works with touch. Speech features depend on mobile browser support.
Q: Is data saved between sessions?▶
Saved compositions use session storage, which persists within a single browser tab session. Closing the tab clears saved items. Game scores also reset on page reload.
Q: Can I add my own poems to the Library?▶
Not through the UI. You can edit the POEMS array in the source code. Each entry needs text, type, typeName, en, fr, source, and century fields.

LIBRARY

Q: Are all the palindromes verified?▶
Yes. Every poem has been tested against stripForCompare + isLetterPalindrome / isWordPalindrome. Stripping removes tashkeel, tatweel, spaces, punctuation, and normalizes alef/hamza variants.
Q: What does the symmetry animation show?▶
It reveals mirror structure. Starting from the center, each pair of letters at positions (i) and (length−1−i) illuminates simultaneously. Gold = match, crimson = mismatch. A vertical gold line marks the axis.
Q: What is the difference between letter and word palindromes?▶
A letter palindrome reads the same character-by-character when reversed (after stripping diacritics). A word palindrome has the same sequence of words when word order is reversed — e.g. "وَدِّعْ سُعَاد وَدِّعْ" reads the same words backwards.

CREATOR

Q: What exactly gets stripped for comparison?▶
Tashkeel (Unicode 0610–061A, 064B–065F, 0670, 06D6–06DC), tatweel (0640), whitespace, punctuation (،؟!,.?), alef variants (إأآ → ا), and hamza carriers (ؤئ → ء). Original text with diacritics is preserved for display.
Q: What does "95% symmetry" mean?▶
The symmetry score counts matching mirror-position letter pairs divided by total pairs. 95% means most pairs match but a few differ. Check crimson letters in the letter map to see which positions to adjust.

GAME

Q: How is scoring calculated?▶
Letters: 10 points. Words: 25 points. Full verse: 50 points. Streak tracks consecutive correct answers and resets on any wrong answer. Timer mode does not affect scoring — if time runs out, the streak resets but no points are deducted.

RECITER

Q: Can I recite my own text?▶
Yes. Paste any Arabic text into the custom text area. It does not need to be a palindrome — the reciter plays any text forward and backward.
Q: What happens in dual mode?▶
Forward pass with gold highlights, 500ms pause, then backward pass with crimson highlights. This audibly demonstrates the palindrome property.

🔧 Help & Troubleshooting

No sound when clicking Listen / Play

Check browser volume is not muted. Try Chrome or Edge for best Arabic TTS support.

Fix: Click anywhere on the page first (browser autoplay policy). Verify Arabic voices with speechSynthesis.getVoices() in the console.

Microphone not working

Speech recognition requires Chrome or Edge. Firefox and Safari do not support the Web Speech Recognition API.

Fix: Grant mic permission when prompted. HTTPS is required on deployed sites; local files usually work.

Arabic keyboard keys look wrong

The keys still function even with fallback fonts. The visual issue is from Google Fonts failing to load.

Fix: Hard-refresh with Ctrl+Shift+R. Check network connectivity for fonts.googleapis.com.

Game shows no puzzle

The first puzzle generates automatically when you switch to the Game tab.

Fix: Click "التالي" (Next) to generate a new puzzle manually.

SVG export looks different from the app

The SVG export is a simplified card format without animations or the letter map.

Fix: This is by design. The SVG is a portable, static snapshot meant for sharing.

Performance issues on mobile

The geometric canvas background animates continuously, which may lag on older devices.

Fix: The animation is purely decorative (12% opacity). To disable: add #geoCanvas { display:none } in dev tools.

Text direction looks wrong

The app sets dir="rtl" globally. If text appears LTR, check browser isn't overriding direction.

Fix: All input fields are explicitly RTL. Ensure no browser extension is interfering.

⚙️ Technical Reference

ARCHITECTURE

Single HTML file (~65KB) containing all CSS, HTML, and JavaScript inline. No modules, no bundler, no framework. CSS custom properties define the design system. One responsive breakpoint at 600px. Four <section> elements toggled by navigation. <canvas> for geometric background.

CORE FUNCTIONS

// Strip diacritics, tatweel, spaces, punctuation, normalize alef/hamza
function stripForCompare(str) {
  return str
    .replace(/[\u0610-\u061A\u064B-\u065F\u0670\u06D6-\u06DC]/g, '')
    .replace(/[\u0640]/g, '')
    .replace(/\s+/g, '')
    .replace(/[،؟!,.?]/g, '')
    .replace(/[إأآا]/g, 'ا')
    .replace(/[ؤئ]/g, 'ء')
    .trim();
}

// Letter-by-letter palindrome check
function isLetterPalindrome(str) {
  const s = stripForCompare(str);
  return s === s.split('').reverse().join('');
}

// Word-by-word palindrome check
function isWordPalindrome(str) {
  const stripped = str.trim().split(/\s+/).map(w => stripForCompare(w));
  const rev = [...stripped].reverse();
  return stripped.join(' ') === rev.join(' ');
}
